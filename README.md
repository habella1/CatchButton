## 기능
1. 기본 도망 기능
  - 마우스가 버튼 위로 이동하면 버튼 위치 변경해서 옮김.
  - 버튼의 현재 X,Y 좌표 실시간으로 표시 (폼 제목에)
  - 버튼 이동 범위 제한
2. 시각적/청각적 피드백
   -효과음 추가 (도망갈 때, 잡았을 때)
     버튼 잡았을 때 버튼 클릭하면 "축하합니다~!" 메시지 박스 출력
3. 점수 표시와 점수별 난이도 조정
  -점수 계산 (버튼 클릭하면 +100, 도망가면 -10
    난이도 조절 (클릭 성공할 때마다 버튼 크기 10%씩 줄임)
4. 게임 오버 및 리셋
  -20번 놓치면 "Game Over" 메시지 출력 + 모든 버튼 비활성화 처리
  -"다시 시작" 버튼 구현 ( 모든 정보 초기화, 게임 재시작)

 
## 구현 시 어려웠던 점 혹은 새로 알게 된 점
- 1단계에서 버튼 가출이 일어나 어려움이 있었는데, ClientSize에서 버튼의 크기만큼 빼서 그 문제를 해결함.
- 3단계에서 효과음을 추가할 때, using System.media; 라이브러리를 가져와야 한다는 것을 알게 됨.
- 3단계에서 버튼 크기를 10%씩 축소할 때, *0.9 를 해줘야한다는 것을 알게 됨.
